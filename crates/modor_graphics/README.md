Graphics module of Modor.

## Usage

The minimum supported version of Rust is 1.60.

To include this library in your project, just add the following line in your `Cargo.toml` file:

```toml
modor = "0.1"
modor_physics = "0.1"
modor_graphics = "0.1"
```
